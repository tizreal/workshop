Coding Workshop: Introduction to Coding for Beginners

About the Workshop

This workshop is tailored to introduce coding to beginners, emphasizing that advanced computer knowledge, mathematical prowess, or a scientific background are not prerequisites for acquiring coding proficiency.

Workshop Activities

  • Generative Art with JavaScript
  • Python for Educators
  • Web-making with HTML & CSS
  • Introduction to SQL

Workshop Details

The workshop will start with the basics of coding and gradually introduce concepts and languages. Experts in coding and programming will lead the sessions, sourced with the assistance of mentors, former lecturers, and friends. The workshop is planned to be intensive and will run in two streams, one during the week and the other at the weekend.

Topics Covered

  • Generative Art with JavaScript
  • Python for Educators
  • Web-making with HTML & CSS
  • Getting started with micro:bit
  • Introduction to SQL and Cybersecurity

Outcome

At the end of the workshop, participants will explore a variety of coding concepts, learn basic techniques translatable to other programming languages, and gain an understanding of the basics of SQL.
